# Cartago Motors

Site vitrine React/Vite pour une concession de véhicules premium.

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir l'URL affichée par Vite.

## Build production

```bash
npm run build
npm run preview
```

## Contenu

- Page d'accueil premium
- Collection de véhicules filtrable
- Fiches véhicules avec modal et vues Profil / Face / Arrière / Intérieur
- Services
- Témoignage
- Contact avec formulaire simulé
- Responsive mobile/tablette/desktop
